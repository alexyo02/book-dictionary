# 📖 BookDictionary

A simple and elegant app to save words you discover while reading — straight into your personal database.

![icon-128.webp](C:\Users\aless\Downloads\bookdictionary\icons\icon-128.webp)

---

## ✨ What it does

While reading a book, you come across a word you don't know (or just love). WordBook lets you:

- 🔍 **Search** the word instantly
- 📝 **Save it** to your personal word database
- 🗂️ **Browse** your collection anytime

All wrapped in a clean, **Material Design** UI.

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/alexyo_02/bookdictionary.git

# Install dependencies
cd bookdictionary
npm install

# Run the app
npm start
```
